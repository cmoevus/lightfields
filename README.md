# Light Fields #

Lighting calculator for horticulture

## Setup ##
1. Enter repository root directory in a terminal shell. The repository root directory is the one containing this README file.
2. Optionally create a Python virtual environment, read these [instructions](http://virtualenvwrapper.readthedocs.io/en/latest/). And add current directory to PYTHONPATH with `$ add2virtualenv .`
3. Run `pip install -r requirements.txt` to install the dependency packages.
4. Run tests `$ pytest`
5. Read these [instructions](http://jupyter-notebook-beginner-guide.readthedocs.io/en/latest/execute.html) for executing notebooks.
